GET /api/interests : liste des intérêts

GET /api/sessions : liste des sessions

GET /api/sessions/{id} : détail de la session portant l'id défini

GET /api/sessions/{id}/comments : commentaires de la session portant l'id défini

POST /api/sessions/{id}/comment : ajoute un commentaire à la session portant l'id défini (Connecté)

GET /api/lightnings : liste des lightning talks

GET /api/lightnings/{id} : détail du lightning talk portant l'id défini

GET /api/sessions/{id}/comments : commentaires du lightning talk portant l'id défini

POST /api/lightnings/{id}/comment : ajoute un commentaire au lightning talk portant l'id défini (Connecté)

GET /api/members : liste des membres (Connecté)

GET /api/members/{id} : détail d'un membre (Connecté)

GET /api/suggestions/sessions : les suggestions de sessions par interêt

GET /api/suggestions/members : les suggestions de membres par interêt
